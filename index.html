labs
00: Summer school (Studio)

Gain experience in the use of the git source code management tool, acquire basic knowledge of the JavaScript language and improve your web application development skill set.
01: Git intro & Donation web app

A basic introduction to the Git versioning system working from the commandline. Specification for a simple dynamic web app named Donation to be implemented using the Play framework.
01: Donation (Solutions)

Sample set of solutions in developing donation app in 5 iterations: authentication through to generating reports.
02: JavaScript Introduction

Beginning a condensed introduction to the JavaScript language with a discussion on data types and some basic features of the language such as identifiers, operators and control flow.
03: Javascript & Maps; MyRent specification.

How to use developer tools to debug into JavaScript code, a discussion on some new features introduced in ES6 such as arrow functions and the spread-rest operator and exploring the somewhat arcane this keyword. Specification for a second dynamic web app named MyRent.
03: MyRent (Solutions)

Sample set of solutions in developing myrent app in 4 iterations: authentication, data capture, reporting and feedback.
04: Map overlays, the DOM & SemanticUI controls

Exploring the use of customised circle and polygon map overlays. Using JavaScript to validate Semantic UI input. Introducing Ajax with some simple applications.
05: WitPress prototype web app, TDD & Advanced Git.

WitPress: A web app to facilitate the creation and management of web logs (blogs). Using a Test Driven Development (TDD) approach in developing a set of entity-related Play models classes. Advanced Git tutorials and presentation building on an the earlier basic introduction.
05: WitPress (Solutions)

Sample set of solutions in developing witpress app in 3 iterations: authentication, post and comment, public blog and feedback.
06: Project Specification & tutorial on Heroku cloud deployment.
