Assignment: The submission process comprises a cover page to be uploaded by means of a dropbox on Moodle. On the page should be links to the three project repositories on Bitbucket and the url to the completed application, deployed on Heroku. A link to download a cover page template is provided in this resource.

Resources

The same MyRent repository used by a student in the labs should be used also for the assignment project.

  • Simply add and tag each story and then push (including tag) to the remote repo on BitBucket.
  • Deploy your finished app to Heroku.

Documentation associated with the assignment as follows:

  • A cover page. You are asked to edit this appropriately and upload in the dropbox that will be provided. Templates are available in Word and pdf formats:
