Welcome

Welcome to the Waterford Institute of Technology (WIT) ICTSkills Summer School, also referred to as ICTSkills Studio. The course spans six consecutive weeks commencing late May and ending early July. The venue is the WIT West Campus at Carriganore. Course content includes the git versioning system, an introduction to the JavaScript language, dynamic web development and a module on oral and written communications. The course is a five credit module. Grading is based on continuous assessment and a web application development project.

Venue

The course will be run on Tuesdays and Wednesdays and industry presentations will take place on Thursdays in the boardroom at Carriganore. You may view maps of the location and route in the next step. Accommodation will also be available elsewhere for work on Mondays and Fridays and for the period between the end of the course and beginning of the autumn semester at a venue or venues to be advised.

Outline

Here we present an outline timeline and the assignment grading rubric.

Basic-Git

Lab describing how to use git to manage a project by creating a local repository, inserting the project files into the repository and creating and managing a tracking copy of the repository on a cloud-based server.

Donation

Develop a dynamic web application called donation that will allow a user to register, log in and log out, make donations and print a report of all donations.

JavaScript-I

Introducing the fundamentals of the javascript language, becoming familiar with some of the different sources of a selection of javascript resources. Explore Google Chrome's javascript-based tools. Understand basic types. Acquire an understanding of the roles that arrays and objects play in a javascript program. All using JetBrains' WebStorm IDE.

JavaScript-II

Examine some of JavaScript's constructs such as conditionals, loops, arrays, objects, functions and scope. Understanding some concepts such as Truthy and Falsy values. simple static web site, include an external JavaScript file and use Google Chrome developer tools to debug into the script at runtime.

JavaScript III (Maps)

Demonstrate how to embed a map in a web page and add a marker or overlay. Enable the marker so that it may be dragged and dropped. Capture the marker's geolocation and dynamically display this on the webpage.

MyRent

myrent, a prototype web application to allow a user to register and authenticate, to gather data on residential properties, to store the data in a database and generate reports on properties located within specified regions.
